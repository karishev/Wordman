Welcome to Wordman!


The main objective of the game is to guess the five letter word.
In the Wordle part, after entering the word, the game colors the letters of the word.
If the letter is gray, there is no such letter in the word.
If the letter is yellow, there is a letter in the word, but in a different position.
If the letter is green, the position is the same in the given word!

In the Hangman part, you enter letters.
If the letter is in the given word, it will appear.
If not, then there is no such letter in the word.
You have only 3 attempts in the Hangman part, no matter if the guess was correct or not.
